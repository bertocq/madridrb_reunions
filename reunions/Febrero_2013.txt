# Videojuegos 2D con Ruby + Competición

Tradicionalmente no se usa Ruby para el desarrollo de videojuegos, pero en esta charla veremos lo fácil que es crear un videojuego 2D con Ruby y también las limitaciones que tiene. Al tener esa base, os voy a proponer una pequeña competición de programación para todos en que se trata de programar el mejor tanque para una batalla entre todos de los participantes. ¡Venid para ver los detalles!

Adam Williams nació en Minneapolis, Minnesota y lleva 6 años viviendo en España. Usa Ruby profesionalmente para hacer desarrollo con Rails y aparte tiene un proyecto personal hecho con Sinatra y el api streaming de Twitter relacionado con el fútbol [http://imbetting.io](http://imbetting.io).
